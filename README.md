# yudin-landing

Статические превью и технические задания для четырех проектов экосистемы доктора Юдина:

- BMJO - 3 страницы;
- АСПЛН - 2 страницы;
- Благотворительный фонд доктора Юдина - 2 страницы;
- ПРОБОС - 1 страница.

В каждом превью показаны рекомендуемые номера стандартных блоков Tilda. Формы работают только в демонстрационном режиме и не отправляют персональные данные.

Основная визуальная система: Manrope, `#3D3D39`, `#3D5445`, `#F6F8F6`, `#F0D9A0`.
